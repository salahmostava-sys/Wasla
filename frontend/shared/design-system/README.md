# Muhimmat UI Design Pattern

Use these shared classes for new screens and large refactors:

- `ds-page`: page root inside `app-content-shell`.
- `ds-toolbar`: filters, search, date pickers, and page actions.
- `ds-section`: primary white panel.
- `ds-section-header`: title and actions inside a panel.
- `ds-section-body`: panel content spacing.
- `ds-table-surface`: table wrapper with horizontal scrolling.
- `ds-control`: custom inputs/select-like controls.
- `ds-chip`: compact status/filter badge.

Surface rule:

- Page canvas is white.
- Panels are white.
- Secondary areas use the very light `muted` or `ds-surface-low` color.
- Borders separate sections; shadows stay subtle.
